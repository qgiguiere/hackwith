About

I created this site as a resource for anyone who has an interest in cybersecurity. There are so many amazing open source tools out there and sometimes it is hard to find which one would be best for you. There are great resources like the awesome lists but I found I was spending a lot of time clicking on links trying to find a good solution for me. So I wanted to create something that would help other people find tools in a more efficient way. Inspiration for this site inclues the awesome lists, The C2 Matrix, The Penetration Testers Framework, and The Red Teaming Toolkit.
